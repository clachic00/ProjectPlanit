# Planit PROJECT
Team Project
`여행 일정 등록 및 커뮤니티 공유 서비스`

## 구현 스택
* Spring Framework
* Spring MVC module
* Mybatis

### 기술 / 구조    
   - 웹 표준 

  `HTML5` `CSS3` `JavaScript` `jQuery` `Bootstrap` `W3C`

- DBMS  - `MySQL`

- `Spring Framework`, `Mybatis`

- API 

   `Rest API` `kakao API` `kakao 주소 API` `kakao map API` `summernote`

- RESTful API 구조

  서버와 클라이언트 통신을 위한 REST 인터페이스 구현

- WAS - `Tomcat8`

- AWS 배포 

  `EC2` `RDS`
